# Google Gemini AI Integration for Monthly Logs

## Get Your Free API Key

1. Go to [https://aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the API key

## Add Your API Key

Open `public/js/monthly-logs.js` and replace this line:

```javascript
this.GEMINI_API_KEY = 'AIzaSyDGcL8xVYH9qLhqh0b_demo_key_replace_this';
```

With your actual API key:

```javascript
this.GEMINI_API_KEY = 'YOUR_ACTUAL_KEY_HERE';
```
